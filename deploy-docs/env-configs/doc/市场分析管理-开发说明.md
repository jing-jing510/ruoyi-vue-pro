# 市场分析管理 - 开发说明

## 创建日期
2025-12-29

## 模块概述
市场分析管理包含三个子模块：
1. **市场分析** - 外链地址（用户自行处理）
2. **综合成本核算** - 附件上传管理
3. **洗选分析报告** - 附件上传管理

## 数据库表结构

### 1. 综合成本核算附件表 (coal_cost_accounting_attach)
- 主键: id
- 报告日期: report_date (date, NOT NULL)
- 报告文件URL: report_file_url (varchar(512), NOT NULL)
- 图片URLs: image_urls (varchar(2000), 多个用逗号分隔)
- 备注: remark (varchar(255))
- 唯一键: uk_tenant_date (tenant_id, report_date)

### 2. 洗选分析报告附件表 (coal_washing_analysis_report_attach)
- 主键: id
- 报告日期: report_date (date, NOT NULL)
- 报告文件URL: report_file_url (varchar(512), NOT NULL)
- 图片URLs: image_urls (varchar(2000), 多个用逗号分隔)
- 备注: remark (varchar(255))
- 唯一键: uk_tenant_date (tenant_id, report_date)

## 开发步骤

### 第一步：创建数据库表
执行 `市场分析管理-新增表SQL.sql` 中的SQL语句

### 第二步：使用代码生成器
1. 登录芋道框架后台
2. 进入代码生成器
3. 导入表：
   - coal_cost_accounting_attach
   - coal_washing_analysis_report_attach
4. 配置生成信息：
   - 模块名: coal
   - 业务名: costAccountingAttach / washingAnalysisReportAttach
   - 类名描述: 综合成本核算附件 / 洗选分析报告附件
   - 作者: 根据实际填写
5. 生成代码并下载

### 第三步：前端调整
参考模块: `yudao-ui/yudao-ui-admin-vue3/src/views/coal/dailyreportattach/`

**重要修改点：**
1. 日期字段格式：
   ```vue
   <el-date-picker
     v-model="formData.reportDate"
     type="date"
     value-format="YYYY-MM-DD"
     placeholder="选择报告日期"
   />
   ```

2. 文件上传组件：
   - 单文件上传: report_file_url
   - 多图片上传: image_urls

3. 表单验证规则：
   - reportDate: 必填
   - reportFileUrl: 必填

### 第四步：后端调整
参考模块: `yudao-module-coal/src/main/java/cn/iocoder/yudao/module/coal/controller/admin/dailyreportattach/`

**注意事项：**
1. LocalDate字段使用 `@DateTimeFormat(pattern = "yyyy-MM-dd")`
2. 唯一键约束处理 (tenant_id + report_date)
3. 文件URL验证

## 菜单配置
需要在系统菜单中添加：
- 市场分析管理（父菜单）
  - 市场分析（外链）
  - 综合成本核算
  - 洗选分析报告

## 注意事项
1. 所有日期字段前端使用 `type="date"` + `value-format="YYYY-MM-DD"`
2. 后端LocalDate字段不需要特殊处理，框架会自动转换
3. 文件上传路径建议：
   - 综合成本核算: `/coal/cost_accounting/`
   - 洗选分析报告: `/coal/washing_analysis/`
4. 唯一键约束：同一租户下，同一日期只能有一条记录
